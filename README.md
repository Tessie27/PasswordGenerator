# Password Generator

A secure and user-friendly password generator application built with PyQt5.

## Features

- Generate standard random passwords
- Generate memorable passwords using word combinations
- Custom password input and validation
- Password strength analysis
- Dark and light themes
- Copy to clipboard functionality
- Save passwords to file
- Secure random generation using `secrets` module

## Requirements

- Python 3.6+
- PyQt5
